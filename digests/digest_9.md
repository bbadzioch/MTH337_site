## Jaimie

:::{epigraph}
What is the difference between loc and iloc, and when is an appropriate time to use each of them? 
:::

In pandas dataframes, `loc` selects rows and columns based on names of rows (given in the dataframe index)
and names of columns. `iloc` also selects rows and columns but based on their numbering (where numbers 
of rows and columns start with 0). Which method is more convenient, depends on circumstances. If you want 
to select every other row in a dataframe `df`, then you can use `df.iloc[::2]`. If you want select rows whose 
indexes are `R1`, `R2`, then you can use `df.loc[["R1", "R2"]]`.

:::

in computer memory, the amount of memory is a limit. However, there are some way 
to optimize this.
