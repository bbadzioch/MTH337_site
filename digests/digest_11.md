## Dixith

:::{epigraph}
What is the main difference between using Matplotlib and Bokeh for graphs, and when should we use each one?
:::


Matplotlib in general produces static plots, so it is useful for created images that will be embedded 
in pdf documents, printed etc., or just plots that need not to be modified after they are created.  
Bokeh produces interactive plots that can be embedded in web pages. Bokeh plots are not very suitable 
for plotting a lot of data - in such cases they can use a lot of computer memory and may be slow to render.
