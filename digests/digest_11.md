## Josh

:::{epigraph}
How difficult would it be to compose a song in Python? In class, we went over how we could play a desired note for a desired period of time - but we also had to click the play button ourselves. Is there a way for the Python code to automatically play our desired sound file, and could we implement it in loops to play a desired pattern, creating a song?
:::

If I understand correctly, you would like to create code that would let you play music and modify it as it was 
playing, by adding new loops etc. This can be done with Python. You would need to build some user interface 
for manipulating audio output. There are libraries for this - e.g. pygame or tkinter. Also, you would probably 
want to use specialized libraries for audio processing, and there are several of these: pydub, pymusiclooper, pyo etc.
I would be an interesting project.
